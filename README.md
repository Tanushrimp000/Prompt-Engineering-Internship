# 🚀 Prompt Engineering Internship Portfolio

Welcome to my Prompt Engineering Internship portfolio! This repository documents the tasks completed during my internship at **SkillCraft Technology**, showcasing practical applications of prompt engineering, creative AI workflows, task automation, and AI assistant design using Large Language Models (LLMs).

## 📌 About the Internship

During this internship, I explored how effective prompt design influences AI-generated outputs by applying techniques such as:

- Prompt Refinement
- Role Prompting
- Context Engineering
- Structured Prompting
- Few-Shot Prompting
- AI Task Automation
- Persona-Based AI Interactions

The objective was to understand how prompt engineering can improve the quality, creativity, consistency, and reliability of responses generated by LLMs.
