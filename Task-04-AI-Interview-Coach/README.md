# Task 04 – AI Interview Coach

## Objective

Develop an AI-powered interview assistant capable of simulating a technical interview through prompt engineering.

## Project Overview

The AI Interview Coach is designed to conduct structured mock interviews by collecting candidate information, generating role-specific technical questions, evaluating responses, providing scores, and offering personalized feedback.

## Features

- Professional interview flow
- Candidate information collection
- Role-based technical questions
- Experience-level customization
- AI-generated evaluation
- Answer scoring
- Strengths and improvement suggestions
- Interview tips
- Adaptive questioning

## Prompt Engineering Techniques Used

- Role Prompting
- Instruction-Based Prompting
- Persona Prompting
- Context Retention
- Multi-turn Conversation Design
- Structured Prompting
- Adaptive Prompting

## Technologies

- Botpress Cloud
- Prompt Engineering
- Large Language Models (LLMs)
